# CareConnect Web Application

CareConnect is a simple full-stack web application that allows users to book medical appointments.  
This project demonstrates real frontendâ€“backend integration using a REST API and MongoDB.

## Tech Stack
- Frontend: HTML, CSS, Vanilla JavaScript
- Backend: Node.js, Express
- Database: MongoDB (Mongoose)

## Features
- Book appointments through a web interface
- Store appointment data in MongoDB
- Fetch and display saved appointments dynamically
- Basic validation and error handling

## How It Works
1. User submits appointment form on the frontend
2. Frontend sends data to Express API
3. Backend stores data in MongoDB
4. Frontend fetches stored data and displays it
